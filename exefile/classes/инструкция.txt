Инструкция по работе с приложением

На экране открытого приложения есть каталог списков и 4 кнопки. Каждая из них отвечает за разные функции:
1) Запустить. Перед нажатием выберите список из каталога путем нажатия один раз левой кнопкой мыши
2) Изменить. Позволяет добавить новое слово в список и его перевод. Перед нажатием так же выбрать список из каталога
3) Добавить. Создает новый список в каталоге, его название и дату создания. Перед нажатием ничего не требуется
4) Удалить. Позволяет удалить список из каталога и базы данных. Перед нажатием выберите список из каталога

Отдельная инструкция по тренировке:
	1) Нажмите кнопку «Начать»
	2) Введите перевод
	3) Нажмите кнопку «Проверить»
	4) Сотрите появившуюся надпись из окна «Введите перевод»
	5) Нажмите кнопку «Продолжить»
